Spring Mobile 1.0.0.M1 (Nov 16, 2010)
-----------------------------------------------------------
The Spring Mobile project provides support for developing cross-platform mobile web applications.
It contains extensions to Spring MVC that aim to simplify mobile application development.

To find out what has changed in this release, see 'changelog.txt'

Please consult the documentation located within the 'docs/reference' directory
of this release and also visit the official Spring Mobile home at:
http://www.springsource.org/spring-mobile

There you will find links to the forum, issue tracker, and other resources.
